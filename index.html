🔬 Méthodologie de Calcul

  • Distribution de Poisson : Modèle probabiliste pour prédire le nombre de buts
  • Moyennes pondérées : Matchs récents avec coefficients plus élevés
  • Facteur domicile/extérieur : Ajustement basé sur les performances locales
  • Analyse offensive/défensive : Comparaison des forces d'attaque et de défense
  • Matrice de probabilités : Calcul de toutes les combinaisons de scores possibles
  • Exclusion des xG : Calculs basés uniquement sur les résultats réels
